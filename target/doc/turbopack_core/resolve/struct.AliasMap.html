pub struct AliasMap<T> { /* private fields */ }
Expand description

A map of AliasPatterns to the [Template]s they resolve to.

If a pattern has a wildcard character (*) within it, it will capture any number of characters, including path separators. The result of the capture will then be passed to the template.

If the pattern does not have a wildcard character, it will only match the exact string, and return the template as-is.
