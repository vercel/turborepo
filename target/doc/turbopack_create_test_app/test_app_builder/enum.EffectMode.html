pub enum EffectMode {
    None,
    Hook,
    Component,
}
Expand description

How to run effects in components.

Variants§

§

None

No effects at all.

§

Hook

As a direct useEffect hook in the component’s body.

§

Component

Rendering an client-side component that has the useEffect hook instead. Good for testing React Server Components, as they can’t use useEffect hooks directly.
