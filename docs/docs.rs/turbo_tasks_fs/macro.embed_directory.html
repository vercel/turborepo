Macro turbo_tasks_fs::embed_directory

source · 
macro_rules! embed_directory {
    ($name:tt, $path:tt) => { ... };
}
Expand description

Returns an embedded filesystem for the given path.

In debug builds, this will create a DiskFileSystemVc.

In release builds, this will embed a directory’s content into the binary and create an EmbeddedFileSystemVc.
