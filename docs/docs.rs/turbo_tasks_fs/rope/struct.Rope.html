Struct turbo_tasks_fs::rope::Rope

source · 
pub struct Rope { /* private fields */ }
Expand description

A Rope provides an efficient structure for sharing bytes/strings between multiple sources. Cloning a Rope is extremely cheap (Arc and usize), and the sharing contents of one Rope can be shared by just cloning an Arc.

Ropes are immutable, in order to construct one see RopeBuilder.
