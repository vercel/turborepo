Enums

SourceMap
The source map spec lists 2 formats, a regular format where a single map covers the entire file, and an “index” sectioned format where multiple maps cover different regions of the file.
Token
A token represents a mapping in a source map. It may either be Synthetic, meaning it was generated by some build tool and doesn’t represent a location in a user-authored source file, or it is Original, meaning it represents a real location in source file.
