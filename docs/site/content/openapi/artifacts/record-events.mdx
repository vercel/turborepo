---
title: Record cache usage events
description: >-
  Records cache usage analytics events. The body of this request is an array of
  cache usage events.


  This endpoint is optional but enables cache hit/miss analytics. Implementers
  can use this data to track cache effectiveness and optimize storage.


  Event types:

  - `HIT`: A cached artifact was found and used

  - `MISS`: No cached artifact was found for the given hash


  Source types:

  - `LOCAL`: The cache event was on the user's local filesystem cache

  - `REMOTE`: The cache event is for the remote cache
